Accessibility Statement

This website is committed to providing equal access to all visitors, regardless of ability or the technology used to access the site. Below is an accurate account of what has been implemented and where limitations remain.

Standard

This website aims to conform to WCAG 2.1 Level AA, developed by the World Wide Web Consortium (W3C). Conformance is partial — the features below have been verified; a full independent audit has not been conducted.

What Is Implemented

  • Skip to main content: A keyboard-accessible skip link appears at the top of every page, allowing screen reader and keyboard users to bypass the navigation.
  • Keyboard navigation: All interactive elements — including the navigation dropdown — can be operated using the keyboard alone. The dropdown opens on focus and closes on Escape.
  • ARIA attributes: The navigation dropdown button carries aria-haspopup, aria-expanded, and aria-label attributes that update dynamically with state.
  • Alternative text: All images carry descriptive alt attributes.
  • Language declaration: Every page declares lang="en" so screen readers use the correct pronunciation rules.
  • Semantic HTML: Pages use <header>, <nav>, <main>, <footer>, and a proper heading hierarchy.
  • Colour contrast: Primary text combinations (navy on off-white, white on navy) meet the WCAG AA 4.5:1 contrast ratio for normal text.
  • Responsive design: The layout adapts to all screen sizes including mobile.

Known Limitations

  • External links (Google Drive PDFs, Google Scholar, etc.) are not under our control and may not meet accessibility standards.
  • The publications filter on the Research page uses JavaScript; users with scripting disabled will see all publications unfiltered, which is a valid fallback.
  • A full independent accessibility audit has not been conducted.
