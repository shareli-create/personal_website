Accessibility Statement

Prof. Shlomo Hareli's website is committed to providing equal access and opportunity to all visitors, regardless of their abilities or the technology they use to access the site.

Our Commitment to Accessibility

We strive to ensure that all content on this website is accessible and usable by everyone, including people with disabilities. This website has been designed and developed with accessibility principles in mind to comply with relevant web accessibility standards.

Standards Compliance

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards, developed by the World Wide Web Consortium (W3C). These guidelines provide recommendations for making web content more accessible to people with disabilities, including those with visual, auditory, mobility, and cognitive disabilities.

Accessible Design Features

  • Semantic HTML: The website uses proper HTML structure to ensure content is meaningful and properly interpreted by assistive technologies.
  • Color Contrast: Text colors have been chosen to provide sufficient contrast against background colors for better readability.
  • Keyboard Navigation: All interactive elements can be accessed using keyboard navigation alone, without requiring a mouse.
  • Alternative Text: Images include descriptive alternative text for screen reader users.
  • Responsive Design: The website is designed to be usable on devices of all sizes, from desktop computers to mobile phones.
  • Clear Navigation: The website has a clear, logical structure with consistent navigation throughout all pages.
  • Readable Fonts: The website uses legible fonts with appropriate sizes to enhance readability.

Assistive Technology Compatibility

This website is compatible with common assistive technologies, including:

  • Screen readers (e.g., NVDA, JAWS)
  • Voice recognition software
  • Text enlargement tools
  • High contrast display settings
  • Keyboard-only navigation

Known Limitations

While we have made significant efforts to ensure accessibility, some limitations may exist. We are committed to addressing any accessibility issues as they are identified.
